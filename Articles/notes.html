Notes

  1. If you want to multiply a number, you need to use the * symbol. For example, 8×8 will not work but 8*8 will work. That is the same when dividing. For example, 8/2 will work but 8÷2 willn't work.
  2. EchoSearch can have unexpected outages, mainly because of updates commitment.
  3. The results section will not be themed until the search appearance changes.
  4. To be able to properly use the conversion feaure, you need to write things like 5lb to kg. If you put things like 5lb to ml, a message will come up as 'Conversion not recognized'. Supported conversions are:
    • Length: Meter (m), Kilometer (km), Centimeter (cm), Millimeter (mm), Inch (in), Foot (ft), Yard (yd), Mile (mi)
    • Weight: Kilogram (kg), Gram (g), Milligram (mg), Pound (lb), Ounce (oz), Ton (ton)
    • Volume: Liter (L), Milliliter (mL), Cubic Meter (m³), Gallon (gal), Quart (qt), Pint (pt)
    • Speed: Meter per Second (m/s), Kilometer per Hour (km/h), Mile per Hour (mph), Foot per Second (ft/s)
    • Time: Second (s), Minute (min), Hour (h), Day (day)
    • Area: Square Meter (m²), Square Kilometer (km²), Square Centimeter (cm²), Square Millimeter (mm²), Acre (acre), Hectare (ha)
    • Currency: US Dollar (USD), Euro (EUR), British Pound (GBP), Japanese Yen (JPY)
