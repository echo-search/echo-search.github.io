How to Install on Android

  1. Download the EchoSearch - Google Play package.zip file to your computer or phone.
  2. Extract (unzip) the file — you’ll find one or more files ending in .apk or .aab.
  3. Transfer the APK or AAB file to your Android device if needed.
  4. On your device, open the file to start installation.
  5. If you see a warning about installing unknown apps, tap “Settings” and allow it just for this install.
  6. Once it finishes, the app will appear on your home screen and in the app drawer.

Tip: If you already installed a previous version, uninstall it before installing the new one.
