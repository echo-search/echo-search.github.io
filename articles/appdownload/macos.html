How to Install with Xcode (on a Mac)

  1. Download the pwabuilder-xcode.zip file to your Mac.
  2. Unzip the file — you’ll find files ending in .xcodeproj and/or .xcworkspace.
  3. If both exist, open the .xcworkspace file (it includes any dependencies like CocoaPods). Otherwise, open the .xcodeproj file.
  4. Xcode will launch. Choose your target device:
    • Simulator — to test on macOS or iOS Simulator.
    • Connected Device — if you have an iPhone or iPad connected.
  5. Click the ▶︎ (Run) button in the Xcode toolbar to build and run the app.
  6. Allow any permissions Xcode requests for signing or trust settings.

Tip: You’ll need Xcode installed from the Mac App Store (free) to open and build the project.

Optional: Export a macOS App (.app or .dmg)

  1. In Xcode, select Product → Archive.
  2. Once the archive is complete, open the Organizer window and choose Distribute App.
  3. Select “Developer ID” (for outside App Store) or “Mac App Store” to export a signed .app.
  4. To create a .dmg, use the Terminal command: 
    hdiutil create -volname "MyApp" -srcfolder "/path/to/MyApp.app" -ov -format UDZO "MyApp.dmg"
