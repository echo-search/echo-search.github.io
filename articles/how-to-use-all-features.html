EchoSearch — User Guide

This document explains how to use all available EchoSearch features (excluding the intentionally omitted ones).

1. Search

  1. Click or tap the search bar.
  2. Type your query.
  3. Press Enter or select the Search button.
  4. Results appear immediately on the page.

2. Search Suggestions

  1. Begin typing in the search bar.
  2. A list of suggestions appears automatically.
  3. Select a suggestion to auto-fill the search bar.
  4. The search runs instantly.

3. Voice Search

  1. Select the microphone button.
  2. Speak clearly.
  3. EchoSearch converts your speech to text and searches it.
  4. If voice recognition is unsupported, the button will be inactive.

4. Surprise Button

  1. Select the Surprise button.
  2. A random joke will appear in a pop-up.

5. Math Solver

EchoSearch can instantly calculate mathematical expressions typed directly into the search bar.

Steps:

  1. Type a maths expression (example: 20 / (3 + 2)).
  2. Press Enter.
  3. EchoSearch displays the result.

Supported:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Brackets

6. Unit Converter

How to use:

  1. Type in the format: value unit to unit (Example: 5 km to m)
  2. Press Enter.
  3. EchoSearch displays the converted value.

Tip: use the short format (e.g. km, cm, mm) rather than the full word (e.g. kilometers, centimeters, millimeters).

Important: Conversions requiring density (e.g., 5 ml to g) are not supported.

Supported Unit Conversions

Length

mm ↔ cm ↔ m ↔ km ↔ in ↔ ft ↔ yd ↔ mi

Area

mm² ↔ cm² ↔ m² ↔ km² ↔ ha ↔ acre

Volume

ml ↔ l ↔ m³ ↔ gal ↔ qt ↔ pint ↔ cup

Mass

mg ↔ g ↔ kg ↔ t ↔ oz ↔ lb

Time

s ↔ min ↔ h ↔ day ↔ week

Speed

m/s ↔ km/h ↔ mph

Data Storage

b ↔ B ↔ kb ↔ mb ↔ gb ↔ tb

Data Transfer Rate

bps ↔ kbps ↔ mbps ↔ gbps

Energy

j ↔ kj ↔ cal ↔ kcal ↔ wh ↔ kwh

Pressure

pa ↔ kpa ↔ bar ↔ psi ↔ atm

Angle

rad ↔ deg

7. Dictionary / Word Meanings

EchoSearch can define words using built-in and online dictionary data.

Steps:

  1. Type a query such as:
    • define sunset
    • meaning of orbit
    • definition of system
  2. Press Enter.
  3. The definition appears instantly.

8. Search History (Session)

  1. Recent searches appear in the history list.
  2. Select a previous query to re-run it.
  3. Select Clear to delete the session’s history.

9. Themes

Preset Themes

  1. Open the Theme Selector dropdown on the home page.
  2. Choose any preset theme.
  3. The theme updates immediately.

Custom Themes

  1. Open the Themes page.
  2. Name your theme.
  3. Pick background, text, and accent colours, and optionally a background image.
  4. Select Save to store the theme.
  5. Your custom themes appear in the list for quick selection.

Editing & Deleting

  • Select Edit to modify a theme.
  • Select Delete to remove a theme.
  • Select Clear All to remove all custom themes.

10. No-JavaScript Fallback

  1. A message appears instructing you to enable JavaScript.
  2. Features remain unavailable until JavaScript is enabled.

11. Troubleshooting

Voice search not working?

Your browser or device might not support speech recognition.

Units not converting?

Check the supported unit list. Density-based conversions (e.g., ml ↔ g) are not supported.

Themes not saving?

Local storage may be disabled in your browser.
