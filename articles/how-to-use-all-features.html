EchoSearch — User Guide

This guide explains how to use all available EchoSearch features.

1. Search

  1. Click or tap the search bar.
  2. Type your query.
  3. Press Enter or click Search.
  4. Results appear instantly.

2. Search Suggestions

  1. Begin typing in the search bar.
  2. Suggestions appear automatically.
  3. Select one or press Enter.

3. Voice Search

  1. Select the microphone button.
  2. Speak clearly.
  3. EchoSearch converts speech to text and searches it.

Note: Voice search requires browser support. Unsupported browsers disable the button automatically.

4. Weather

EchoSearch provides real-time weather using natural queries.

Examples:

  • weather in london
  • weather tomorrow in dubai

Shows current conditions, wind speed, and upcoming hourly forecast.

5. Time & Date

Examples:

  • time in tokyo
  • today's date

6. Translation

How to use:

  • hello in spanish
  • who are you in arabic

Source language is automatically detected.

7. Dictionary / Word Meanings

Examples:

  • define sunset
  • meaning of orbit
  • definition of system

Pronunciation audio is included when available.

8. Math Solver

Steps:

  1. Type a maths expression (example: 20 / (3 + 2))
  2. Press Enter.
  3. EchoSearch displays the result and opens the calculator.

Supported:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Brackets
  • Powers

9. Unit Converter

How to use:

  1. Type in the format: value unit to unit (Example: 5 km to m)
  2. Press Enter.
  3. EchoSearch displays the converted value.

Tip: use the short format (e.g. km, cm, mm) rather than full words.

Important: Conversions requiring density (e.g., 5 ml to g) are not supported.

Supported Unit Conversions

Length

mm ↔ cm ↔ m ↔ km ↔ in ↔ ft ↔ yd ↔ mi

Area

mm² ↔ cm² ↔ m² ↔ km² ↔ ha ↔ acre

Volume

ml ↔ l ↔ m³ ↔ gal ↔ qt ↔ pint ↔ cup

Mass

mg ↔ g ↔ kg ↔ t ↔ oz ↔ lb

Time

s ↔ min ↔ h ↔ day ↔ week

Speed

m/s ↔ km/h ↔ mph

Data Storage

b ↔ B ↔ kb ↔ mb ↔ gb ↔ tb

Data Transfer Rate

bps ↔ kbps ↔ mbps ↔ gbps

Energy

j ↔ kj ↔ cal ↔ kcal ↔ wh ↔ kwh

Pressure

pa ↔ kpa ↔ bar ↔ psi ↔ atm

Angle

rad ↔ deg

10. Direct Website & Domain Search

  • Typing example.com opens the site directly.
  • site:example.com searches inside that domain.

11. Search History

  1. Recent searches appear automatically.
  2. Select one to re-run it.
  3. Select Clear to remove history.

12. Themes

Preset Themes

Select built-in visual themes instantly.

Custom Themes

  1. Create and save your own colour or image themes.
  2. Edit or delete themes anytime.

13. Surprise Button

Displays a random joke in a pop-up.

14. No-JavaScript Fallback

EchoSearch requires JavaScript. A message will appear if it is disabled.

15. Troubleshooting

Voice search not working?

Your browser may not support speech recognition.

Units not converting?

Check the supported unit list above.

Themes not saving?

Local storage may be disabled.
