Descarga la versión del proyecto Xcode de la aplicación web progresiva de EchoSearch a continuación. El archivo ZIP incluye tanto .xcodeproj como .xcworkspace para usar con Xcode en macOS.

Descargar para macOS (.zip)

Cómo instalar con Xcode (en Mac)

  1. Descarga el archivo pwabuilder-xcode.zip en tu Mac.
  2. Descomprime el archivo — encontrarás archivos que terminan en .xcodeproj y .xcworkspace.
  3. Abre el archivo .xcworkspace (incluye dependencias como CocoaPods). Si no, abre el archivo .xcodeproj.
  4. Xcode se abrirá. Selecciona el simulador.
  5. Haz clic en el botón ▶︎ (Run) en la barra de herramientas de Xcode para compilar y ejecutar la app.
  6. Acepta cualquier permiso que Xcode solicite para firma o confianza.

Consejo: Necesitarás tener Xcode instalado desde la App Store (gratis) para abrir y compilar el proyecto.

Opcional: Exportar una app de macOS (.app o .dmg)

  1. En Xcode, selecciona Product → Archive.
  2. Cuando termine, abre la ventana Organizer y selecciona Distribute App.
  3. Elige “Developer ID” (fuera de App Store) o “Mac App Store” para exportar un .app firmado.
  4. Para crear un .dmg, usa este comando en Terminal: 
    hdiutil create -volname "MyApp" -srcfolder "/path/to/MyApp.app" -ov -format UDZO "MyApp.dmg"
